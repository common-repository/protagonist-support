Neem vanuit WordPress direct contact op met de support-afdeling van Protagonist.

== Description ==

De Protagonist Support plugin activeert een venster in het dashboard van uw WordPress installatie, waarmee u direct in contact kunt komen met de Protagonist support-afdeling.

== Installation ==

1. Upload de map `protagonist_support` naar de `/wp-content/plugins/` map
2. Activeer de plugin via het 'Plugins' menu in het Wordpress Dashboard

Veel plezier met de Protagonist Support plugin voor Wordpress!
